## Description
Ce projet consiste à créer une API RESTful simple pour gérer une liste de tâches (to-do list) en utilisant Node.js et Express. L'API permet de créer, lire, mettre à jour et supprimer des tâches. Le projet inclut également des tests automatisés écrits en Python pour vérifier le bon fonctionnement de l'API.
## Fonctionnalités
- Créer une tâche
- Lire toutes les tâches
- Mettre à jour une tâche
- Supprimer une tâche
