Creation-API

Création d’une API Node.js + Collaboration GitHub + Tests Python

Description

Ce projet consiste à créer une API RESTful simple pour gérer une liste de tâches (to-do list) en utilisant Node.js et Express.

L'API permet de créer, lire, mettre à jour et supprimer des tâches.
Le projet inclut également des tests automatisés en Python pour vérifier le bon fonctionnement de l'API.

Fonctionnalités

Créer une tâche

Lire toutes les tâches

Mettre à jour une tâche

Supprimer une tâche

Technologies utilisées

- Node.js

- Express

- Supabase (base de données)
- Python (tests automatisés)

Installation

Cloner le dépôt GitHub :

git clone https://github.com/ton-utilisateur/creation-API.git
cd creation-API/mini-api-todo


Créer l'environnement virtuel pour Python :

python -m venv venv


Sur Windows :

venv\Scripts\activate


Sur Linux / Mac :

source venv/bin/activate


Installer les dépendances Node.js :

npm install


Configurer les variables d'environnement :

Crée un fichier .env à la racine du projet et ajoute les informations suivantes :

SUPABASE_URL=your_supabase_url
SUPABASE_SERVICE_KEY=your_supabase_service_key


Remplace your_supabase_url et your_supabase_service_key par les valeurs de ton projet Supabase.

Démarrer le serveur :

npm start


L'API sera accessible à l'adresse :

http://localhost:3000
